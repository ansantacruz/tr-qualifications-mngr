# tr-qualifications-mngr

Microservice that manages the grades module.

## Install:

 - [ ] Clone that repo: https://github.com/ansantacruz/tr-qualifications-mngr.git
 - [ ] Execute the next commands:
	        
    `nvm install 16`

    `nvm use 16`

    `npm i`

    `npm start`
